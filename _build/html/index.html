Runs entire auto focus algorithm in current XY position. Gives back predicted in focus z position via focus_score method which is the Brenner score.

Parameters

Returns

z coordinate for in focus plane

Return type

float

Takes focus scores and its associated z and fits data with gaussian. Gives back position of the fitted gaussian’s middle (x0 parameter) which is the ideal/ in focus z plane

Parameters

Results

z coordinate for in focus plane

float
